# 🧪 PyThermoDB

PyThermoDB provides tools to inspect databooks and tables, build thermodynamic
property objects from tabular references, and package those objects into reusable
ThermoDB files.

## 📦 Install

```bash
pip install pyThermoDB
```

```python
import pyThermoDB as ptdb
print(ptdb.__version__)
```
